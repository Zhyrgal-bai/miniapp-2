# Быстрый старт для мерчанта

Краткое руководство: от заявки до первого заказа.

## 1. Регистрация

1. Откройте Mini App платформы → **Мои магазины** (`/merchant`)
2. Нажмите **Создать** и заполните форму:
   - Название магазина
   - Токен бота (от [@BotFather](https://t.me/BotFather))
   - Телефон для связи
   - *(опционально)* **API Key** и **Account ID** Finik — можно указать сразу или позже в настройках
3. Дождитесь одобрения оператором (обычно до 48 часов)

## 2. Настройка оплаты (Finik)

1. В карточке магазина → **Настройки**
2. Введите **API Key** и **Account ID** Finik (оба обязательны для приёма оплаты)
3. При необходимости укажите legacy **Secret** для текущего HTTP API
4. Сохраните и проверьте статус **Webhook OK**

Без Finik покупатели не смогут оплатить заказ онлайн.

## 3. Каталог

1. Откройте магазин → **Админка** → **Товары**
2. Добавьте минимум **5 товаров** с фото и ценами
3. Создайте **категории** для навигации

## 4. Витрина

1. **Дизайн** → настройте баннер, цвета, блоки
2. Нажмите **Опубликовать**
3. Откройте витрину по ссылке `/s/ваш-slug` или из карточки магазина

**Готовность магазина** отображается на главной платформы — следуйте рекомендациям до 100%.

## 5. Первый заказ

1. Откройте витрину как покупатель (второй Telegram-аккаунт или попросите друга)
2. Добавьте товар → оформите заказ → оплатите
3. В **Заказы** смените статус (принят → отправлен → доставлен)

## 6. Поддержка

- Покупатели пишут через раздел **Поддержка** в витрине
- Отвечайте в **Админка → Поддержка**
- Используйте **быстрые ответы** для типовых сообщений

## 7. Аналитика

**Операции** в админке показывает:
- выручку и заказы за период
- конверсию и посетителей
- инсайты и рекомендации

## Обратная связь (beta)

На странице **Мои магазины** внизу есть форма **Обратная связь** — сообщайте о багах и непонятных экранах. Это помогает улучшить продукт.

## Частые проблемы

| Проблема | Решение |
|----------|---------|
| Webhook ❌ | Проверьте токен бота, нажмите «Проверить webhook» |
| Витрина не открывается | Опубликуйте дизайн; проверьте подписку |
| Нет оплаты | Настройте Finik в настройках магазина |
| Заказ не приходит | Проверьте уведомления; раздел Заказы в админке |
